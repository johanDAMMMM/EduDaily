# Edu-Daily English V21.0
**Autor:** Johan David Arguello Medellín
**Curso:** 4to Semestre - Ingeniería de Sistemas UNAD

## Descripción
Una aplicación interactiva diseñada para el aprendizaje de inglés cotidiano. El software implementa una progresión lógica desde vocabulario básico (A1) hasta estructuras gramaticales complejas (B2) como el presente simple y el uso del verbo *To Be*.

## Características
* **4 Niveles de aprendizaje:** A1, A2, B1 y B2.
* **Densidad académica:** 10 lecciones y 10 preguntas por nivel (40 temas en total).
* **Interfaz Responsiva:** Uso de `customtkinter` con contenedores desplazables (scroll) para asegurar que todo el contenido sea visible.
